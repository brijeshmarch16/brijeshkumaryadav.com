## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- next-themes
- Vercel Analytics

## Running the Project

1. Clone the repository.
2. Install dependencies with `pnpm install`.
3. Start the development server with `pnpm dev`.
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
